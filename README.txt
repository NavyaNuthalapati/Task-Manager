Project Title: Team Task Manager

Description:
This is a simple front-end Team Task Manager created using HTML, CSS, and JavaScript. 
It allows a user to sign up, log in, and then view a basic dashboard showing total projects and tasks.

Features:
- Sign Up with name, email, password, and role (Admin or Member).
- Login with email and password.
- Shows login status in the header (Not logged in / Logged in as Name (Role)).
- Displays a Dashboard section after successful login.
- Simple, responsive UI with a soft pastel theme.

Technologies Used:
- HTML: for page structure (header, login form, signup form, dashboard, footer).
- CSS: for styling, layout, and responsive design.
- JavaScript: for signup and login logic, storing users in an array, 
  checking credentials, and updating the UI based on login status.

How to Run:
1. Download all project files (index.html, style.css, app.js).
2. Keep all files in the same folder.
3. Open index.html in a web browser.
